# Heart-Failure-Predictor
Machine learning project for heart failure prediction using clinical data. Includes preprocessing, exploratory analysis, feature engineering, and supervised model comparison.

The dataset being used is the same avaible for download at (1).

- (1): Dataset: Fedesoriano. Heart Failure Prediction Dataset. Kaggle, 2021. Available at: https://www.kaggle.com/datasets/fedesoriano/heart-failure-prediction
